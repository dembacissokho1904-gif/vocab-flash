# Vocab Flash

Vocab Flash is a versatile application designed for learning and revising vocabulary through flashcards. It is available as both a website and a mobile app, ensuring users can study their vocabulary anytime, anywhere.

## Installation Instructions

### Website
1. Clone the repository:
   ```bash
   git clone https://github.com/dembacissokho1904-gif/vocab-flash.git
   ```
2. Navigate into the project directory:
   ```bash
   cd vocab-flash
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```
5. Access the application in your web browser at `http://localhost:3000`

### Mobile App
1. Ensure you have the Flutter environment set up on your machine.
2. Clone the repository:
   ```bash
   git clone https://github.com/dembacissokho1904-gif/vocab-flash.git
   ```
3. Navigate into the project directory:
   ```bash
   cd vocab-flash/mobile
   ```
4. Get the Flutter dependencies:
   ```bash
   flutter pub get
   ```
5. Run the app:
   ```bash
   flutter run
   ```

## Features
- **Customizable Flashcards**: Users can create and personalize their own flashcards.
- **Rich Media Support**: Add images and audio to flashcards for enhanced learning.
- **Track Progress**: Monitor learning progress through built-in analytics.
- **Offline Access**: Study without an internet connection.
- **Multi-Device Sync**: Synchronize progress across multiple devices.

## Deployment Options
- **Website**: Deploy using services like Vercel, Netlify, or your own server.
- **Mobile App**: Publish on Google Play Store and Apple App Store after ensuring compliance with their guidelines.
